# Golden Trilar Factory – Landing Page

A professional landing page designed for **Golden Trilar Factory (مصنع الهياكل الذهبية)**, specializing in manufacturing metal structures and trailers.  
The page serves as a digital contact hub, allowing customers to quickly reach the factory through phone calls, social media platforms, WhatsApp, and Google Maps.

---

## 🏭 Project Description

This project is a **static landing page** built to represent an industrial factory brand online.  
It focuses on simplicity, fast redirection, and easy access to all official communication channels.

The page automatically redirects visitors to the official website while still providing quick-access links for direct communication.

## 🛠️ Technologies Used

- HTML5  
- CSS3

  - Facebook

  - Twitter (X)
  - Instagram
  - TikTok
  - Snapchat
- Google Maps location link
